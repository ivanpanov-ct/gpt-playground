### Setup 
- create an OpenAI API key https://platform.openai.com/api-keys 
- create file '.env'
- put there OPENAI_API_KEY="${your OpenAI API key}"
- create an assistant here https://platform.openai.com/assistants 
- 

### Start
- execute 'node webserver.js'
